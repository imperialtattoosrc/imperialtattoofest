# imperialtattoofest
O projeto Imperial Tattoo Contest consiste na realização de um evento cultural de três dias focado na celebração do rock and roll e na arte da tatuagem. O evento contará com shows de 9 bandas independentes (3 por dia), concurso de tatuagem, praça de alimentação e um Espaço Kids, oferecendo uma programação voltada para toda a família.
